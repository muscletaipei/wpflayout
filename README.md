## Feature

## Use
Visual Studio
## 常用屬性
1. Margin：控件對外的邊界
2. Padding ：控件內容的邊界
3. HorizontalAlignment：控件的水平對齊
4. VerticalAlignment：控件的垂直對齊
5. HorizontalContentAlignment：控件內容的水平對齊
6. VerticalContentAlignment：控件內容的垂直對齊
7. MinWidth, MaxWidth：控件的 最大\最小 寬度
8. MinHeight, MaxHeight：控件的 最大\最小 高度
